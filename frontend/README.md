# Vaazhi React UI

Put your Supabase URL and anon key into `frontend/.env`.

## Variables

- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`

## Run

1. Install dependencies in `frontend`
2. Run `npm start`
3. Open the localhost URL shown in the terminal
4. Use the form to add income, expense, and transfer rows into `ledger_entries`
